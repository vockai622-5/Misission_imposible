# Импорт студентов в T-FLEX DOCs

Консольное приложение для загрузки данных студентов из Excel и автоматического создания в T-FLEX DOCs:

- группы пользователей
- карточек пользователей
- рабочих папок студентов в справочнике **«Файлы»**

## Что делает программа

1. Считывает Excel-файл со списком студентов
2. Обрабатывает и валидирует данные
3. Создаёт группу в T-FLEX DOCs
4. Создаёт пользователей по данным из Excel
5. Создаёт папку группы в справочнике **«Файлы»**
6. Создаёт отдельную рабочую папку для каждого студента

## Требования

- Windows
- установленный **T-FLEX DOCs 17**
- доступ к серверу T-FLEX DOCs
- корректно настроенные библиотеки `TFlex.DOCs.*`
- Excel-файл с данными студентов
